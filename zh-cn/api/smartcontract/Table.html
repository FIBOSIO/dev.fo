table

对象 Table

multi index table 对象

成员属性

name

table 名

类型：String

实例：

code

指向合约发布者的名称

类型：String

实例：

scope

table 中数据所属的 account_name

类型：String

indexes

查询当前索引，返回所有索引对象，每个索引是一个新的 Table 对象

类型：Object

成员函数

emplace

向 table 存入新数据

调用参数:

  • payer: String, 为此次操作付费的账户
  • val: Object, 将要存入到 table 的值

实例：

find

从 table 查找数据

调用参数:

  • id: Value, 查询的参数

返回结果：

  • DBIterator，返回一个数据库迭代器

实例:

get_primary_key

生成自增主键

实例:

lowerbound

从 table 查找小于参数结果

调用参数:

  • id: Value, 查询的参数

返回结果：

  • DBIterator，返回一个数据库迭代器

实例:

upperbound

从 table 查找大于参数结果

调用参数:

  • id: Value, 查询的参数

返回结果：

  • DBIterator，返回一个数据库迭代器

实例:

toString

返回对象的字符串表示，一般返回 “[Native Object]”，对象可以根据自己的特性重新实现

返回结果:

  • String, 返回对象的字符串表示

实例：

toJSON

返回对象的 JSON 格式表示，一般返回对象定义的可读属性集合

调用参数:

  • key: String, 未使用

返回结果:

  • Value, 返回包含可 JSON 序列化的值
