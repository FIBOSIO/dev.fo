Msig Contract

msig 是 multiple signature（多重签名）的简写，顾名思义，就是让多个账户对一起事务进行签名。可以异步提出、批准、发布经过多方同意的事务。多重签名是根据账户或公钥所拥有权限的权重来决定的，权重达到阈值才能签名成功。

propose

name type description
proposer string 提案人
proposal_name string 提案名
requested string 提案通过所需权限
